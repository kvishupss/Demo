Instructions

Prerequisites
Make an account on Git Hub/ Git Lab
Install Git on local Machine
Install VSCode on your local machine
Install CLI on your local machine
Initial Steps in Git Hub
Click "+" sign on top right corner / Click "Git icon" at top left corner
Click on Create a new Repository.
Give your repository a name.
Create a file. (Ex.ReadMe.md)
Fill something in body
Commit the changes with some message.

Congratulations! Your first version is onboard.

Cloning your repository
Create a folder in your local machine to keep the files of your repository.
Open VSCode
Open terminal by using Task Bar or (CTRL + `)
Check the Git version by using command "git version"
Click on file in menu located in top left side, then open folder
Select your folder which you just created in your local machine
You will see that same folder's path in the terminal
Go to Git Hub, open your ReadMe.md file
click on code tab, there you will find a (HTTPS / SSH/ GIT CLI) URL, copy any one of them
Paste this copied url in your command terminal of VSCode, using command "git clone "URL", Press Enter
You will see your file in the Explorer of VSCode now

Cloning of Repository is successful on your Local Machine.

Authenticate GitHub account and Generate an SSH key
Open Terminal
Configure your account by using command (git config --global user.email "Email Address") / (git config --global user.name "name")
Select open in web browser and enter the password for authentication
Key Generation
Open terminal in VSCode and enter (ssh-keygen -t rsa -b 4096 -C "Email Address"), Press Enter
Give your key a name (Ex. testkey or demokey or repokey)
Neglect the passphrase for now and press enter 2 time
You will see some pattern shown in your terminal
In terminal write (ls) , press enter
All the available files in your repository will be shown in the directory
Check if your keyname is there or not, there will be two files (keyname & keyname.pub)
Do not share these files with anyone.
To fetch the key write (cat keynam.pub)
You will the a big string showed up which is starting from ssh and ending on your email address
Copy that complete String, avoid copying spaces from start and end
Open Git Hub account.
Click your profile picture --> Click on Settings --> Click on SSH & GPG keys -->
--> Click on NEW SSH Key --> Paste your key in the box --> Save it.

Succefully authenicated your agent.

Adding a new file to your repository using Local Machine
Open VSCode, In the explorer move your cursor on folder name you'll find a new file button
Create a new file of any type
Give it some Body
CTRL +S to save it
Open Terminal (CTRL =`)
Write (git status)
It will show how many files have been added to folder in local machine.
Write (git add .), to add all the new files at once and update the previous one
Or write (git add filename.extension), to add specefic file press enter
Now to commit the changes made by you write (git commit -m "Update or create message" -m "Some Description")
On successful commit, use the push command
Write (git push origin main), press enter.

Successfully added/ updated the file from local machine.

Go to the Git Hub and refresh you repository page, you'll find the new file in there with a comment and datetime stamp.
